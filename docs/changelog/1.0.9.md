## Problema risolto: GR1 — Sinonimo `'porta'` duplicato su 5 item

### Analisi

In `game/rooms/corridoioPrincipale.ts`, cinque item diversi condividevano il sinonimo generico `'porta'`:

| Item ID | Sinonimo problematico |
|---|---|
| `porta_nord` | `'porta'` |
| `porta_sud` | `'porta'` |
| `porta_ovest` | `'porta'`, `'grande porta'` |
| `porta_est` | `'porta'`, `'entrata'` |
| `porta_laterale` | `'porta'`, `'alloggi'`, `'dormitori'` |

Il sistema di risoluzione degli item in `gameLogic.ts` usa `Array.find()`: restituisce il **primo** match. Quando il giocatore digita `ESAMINA PORTA`, il sistema trova sempre `porta_nord` — le altre quattro porte diventano inaccessibili tramite sinonimo generico, rendendo il gioco inconsistente.

### Correzione

**Sinonimi aggiornati:**

| Item ID | Prima | Dopo |
|---|---|---|
| `porta_nord` | `['porta']` | `['porta a nord', 'porta spirale nord']` |
| `porta_sud` | `['porta']` | `['porta a sud', 'porta spirale sud']` |
| `porta_ovest` | `['porta', 'grande porta']` | `['porta a ovest', 'porta ovest', 'grande porta']` |
| `porta_est` | `['porta', 'entrata']` | `['porta a est', 'porta est', 'entrata']` |
| `porta_laterale` | `['porta', 'alloggi', 'dormitori']` | `['alloggi', 'dormitori']` |

**Comando aggregato aggiunto:**

Aggiunto nella lista `commands` il regex `^(esamina|guarda|analizza) porta$` che risponde con un testo orientativo che elenca le quattro porte e invita il giocatore a usare i comandi specifici:

```
"Ci sono diverse porte: una a NORD, una a SUD, una grande porta a OVEST e quella
da cui sei entrato a EST. Usa 'esamina porta nord', 'esamina porta sud',
'esamina porta ovest' o 'esamina porta est' per esaminarle singolarmente."
```

Questo è l'approccio standard delle avventure testuali classiche: quando un nome è ambiguo, il gioco chiede disambiguazione invece di scegliere arbitrariamente.

### Verifica

- `npx tsc --noEmit` → zero errori ✅
- `ESAMINA PORTA NORD`, `ESAMINA PORTA SUD`, `ESAMINA PORTA OVEST`, `ESAMINA PORTA EST` → ciascuna descrive l'item corretto ✅
- `ESAMINA PORTA` → mostra il testo orientativo (non più la porta nord arbitrariamente) ✅
- `ESAMINA GRANDE PORTA` → ancora `porta_ovest` ✅
- `ESAMINA ENTRATA` → ancora `porta_est` ✅
- `ESAMINA ALLOGGI` → ancora `porta_laterale` ✅
