# Versione 0.4.6

## Correzione Build per Sottocartelle

- **Problema:** L'applicazione risultava in una schermata vuota se distribuita all'interno di una sottocartella di un dominio (es. `miosito.com/gioco/`).

- **Causa Iniziale:** Il processo di build di Vite generava percorsi assoluti per le risorse (JS, CSS).
- **Soluzione 1:** È stato modificato il file `vite.config.ts` aggiungendo l'opzione `base: './'`.

- **Causa Secondaria:** Anche dopo la prima correzione, il problema persisteva. Un'analisi più approfondita ha rivelato che il file `index.html` sorgente conteneva percorsi hardcoded assoluti per lo script principale e l'icona (es. `src="/index.tsx"`).
- **Soluzione 2:** Sono stati corretti i percorsi nel file `index.html` sorgente da assoluti a relativi (es. `src="./index.tsx"`).

Questa doppia correzione assicura che sia il processo di build sia i file sorgente siano configurati per una distribuzione corretta in qualsiasi sottocartella.
