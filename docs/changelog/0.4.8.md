# Changelog v0.4.8 - Refactoring Core Engine (Fase 1)

## Panoramica
Questa versione introduce il nuovo motore di gioco basato su oggetti ("Entity-Component lite"), abbandonando il vecchio sistema basato puramente su regex per le interazioni base.

## Modifiche Implementate

### 1. Core Engine (Refactoring)
- **Nuova Interfaccia `Item`:** Definita in `types.ts`, supporta proprietà come `id`, `name`, `description`, `details` (per scanner), `isPickable`, `isFixed`, `onUse`, `onCombine`.
- **Aggiornamento `Room`:** Le stanze ora contengono un array `items` che definisce gli oggetti interagibili presenti.
- **Parser Ibrido (`gameLogic.ts`):**
    - Il parser ora scansiona prima gli `items` della stanza corrente per comandi come `ESAMINA`, `ANALIZZA`, `PRENDI`, `USA`.
    - Supporto per sinonimi multipli.
    - Gestione automatica dell'inventario per oggetti `isPickable`.

### 2. Conversione Stanze (Iniziale)
- **Plancia della Santa Maria:** Convertita al nuovo sistema. Oggetti: `Nave Aliena`, `Pannello di Controllo`, `Radio`.
- **Stiva:** Convertita. Oggetti: `Tuta Spaziale`, `Kit di Manutenzione`, `Casse di Minerale`, `Portello Esterno`.

### 3. UI
- Migliorato il feedback visivo per le azioni di raccolta e analisi.
