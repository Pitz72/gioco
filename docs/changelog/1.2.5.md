## CI/CD — Build cross-platform ufficiale (2026-03-28)

### macOS e Linux ora versioni ufficiali

Implementata pipeline **GitHub Actions** (`.github/workflows/build-release.yml`) per la compilazione automatica cross-platform. A partire da questa versione, **macOS e Linux sono piattaforme di distribuzione ufficiali** al pari di Windows.

**File prodotti per ogni release:**

| Piattaforma | Formato | Note |
|---|---|---|
| 🪟 Windows | `.exe` (NSIS installer) | Installazione guidata |
| 🐧 Linux | `.AppImage` | Portabile, nessuna installazione richiesta |
| 🍎 macOS | `.dmg` | Non firmato Apple ID — tasto destro → Apri |

**Dettagli tecnici:**
- Trigger: `git push origin vX.Y.Z` crea automaticamente la GitHub Release con tutti e 3 i file allegati
- Trigger manuale disponibile dal pannello GitHub Actions
- `extraFiles` configurato per-piattaforma (LEGGIMI.txt in win/linux; escluso dal .app bundle macOS per compatibilità firma)
- `hardenedRuntime: false` su mac per firma ad-hoc senza Apple Developer ID
- `artifactName` standardizzato: `IlRelittoSilente-${version}-${os}.${ext}`
- Save/load verificato cross-platform: `app.getPath('userData')` via IPC Electron gestisce i percorsi nativi per ogni SO

**Percorsi salvataggi per SO:**

| SO | Percorso salvataggi |
|---|---|
| Windows | `%APPDATA%\Il Relitto Silente\saves\` |
| macOS | `~/Library/Application Support/Il Relitto Silente/saves/` |
| Linux | `~/.config/Il Relitto Silente/saves/` |

> I save file non sono portabili tra piattaforme diverse (percorsi SO-specifici, by design).
