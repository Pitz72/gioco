# v1.1.34 — Bugfix: N2 monologo bypassabile via alias corpo (arcaBiologica.ts)

## Bug

Il comando di stanza introdotto in v1.1.26 (N2) intercettava ESAMINA CADAVERE
con questo regex:

```
^(esamina|guarda) (cadavere|corpo|figura|astronauta)$
```

In arcaBiologica.ts esiste già un secondo handler (ESAMINA EXTRA) che copre:
`forma`, `forma immobile`, `resti`, `alieno`, `creatura`

Queste parole compaiono nella descrizione della stanza ("figura immobile
accasciata contro una delle capsule"). Se il giocatore usava uno di questi
alias come primo esame, il secondo handler intercettava prima che il flag
`cadavereEsaminatoFirst` fosse impostato, mostrando la descrizione standard
senza il monologo di N2. Il beat emotivo del ritrovamento veniva saltato.

## Fix

Esteso il regex del comando N2 per assorbire tutti gli alias del corpo:

```
^(esamina|guarda) (cadavere|corpo|figura|forma|forma immobile|resti|astronauta|alieno|creatura)$
```

Il secondo handler (ESAMINA EXTRA) rimane invariato per le visite successive
al primo, dove il flag è già impostato e il nostro handler restituisce la
descrizione standard — identica al comportamento precedente.
