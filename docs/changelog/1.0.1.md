# Changelog v1.0.1 - Desktop Release

## 1. Packaging Desktop (Electron)
Questa versione segna il debutto dell'applicazione come software desktop standalone.

- **Integrazione Electron:** Implementato il wrapper Electron per eseguire il gioco come applicazione nativa.
- **Main Process:** Configurato `electron/main.ts` per gestire il ciclo di vita della finestra, la sicurezza (Context Isolation) e il caricamento condizionale (Dev vs Prod).
- **Build System:** Aggiornato `package.json` con script dedicati (`electron:dev`, `electron:build`) e configurazione `electron-builder`.
- **Installer Windows:** Generazione automatica di installer `.exe` (NSIS) ottimizzati.
- **Supporto Multi-Piattaforma:** Predisposta configurazione per build macOS (DMG) e Linux (AppImage), con le dovute limitazioni di firma digitale se compilato da Windows.
- **Bugfix:** Risolto conflitto ESM/CommonJS nel processo principale Electron tramite script di build personalizzato (`electron/build-helper.js`).

## 2. Note Tecniche
- Il gioco ora gira in un container Chromium isolato, mantenendo l'estetica "monitor a fosfori verdi" senza l'interfaccia del browser.
- Aggiunta gestione tastiera per uscita rapida (`ESC`) e ritorno al menu (`M`) nella schermata di Game Over, specifica per l'esperienza desktop.
