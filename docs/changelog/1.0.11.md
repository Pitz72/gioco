# Changelog v1.0.11 — Alias `INDOSSA` → `USA` in normalizeCommand

**Data:** 2026-03-20
**Tipo:** Fix criticità media (M1)
**Categoria:** UX / Parser comandi

---

## Problema risolto: M1 — `INDOSSA [oggetto]` produce errore confuso

### Analisi

Il MANUALE OPERATIVO include `INDOSSA [oggetto]` nella tabella comandi. Il gioco richiede `INDOSSA TUTA SPAZIALE` come primo passo reale dell'avventura (senza tuta non si può uscire nella Stiva). Tuttavia il verb set del sistema item era:

```typescript
const match = normalizedCommand.match(/^(esamina|guarda|analizza|prendi|raccogli|usa|attiva) (.+)$/);
```

`indossa` non era incluso. Digitando `INDOSSA TUTA SPAZIALE`, il comando cadeva nel generic fallback (riga ~280) che restituiva:

```
"Cosa vuoi indossare con 'tuta spaziale'? Devi essere più specifico (es. USA TUTA SPAZIALE SU ...)."
```

Questo messaggio è fuorviante: suggerisce `USA X SU Y` per un'azione che non richiede bersaglio, disorientando il giocatore all'ingresso dell'avventura.

### Correzione

Aggiunto un alias in `normalizeCommand()`, nella sezione "Azioni", che converte `INDOSSA X` → `USA X` prima che il comando raggiunga il dispatcher:

```typescript
// Alias verbi: INDOSSA → USA (stessa semantica nel contesto di gioco)
if (normalized.startsWith('indossa ')) {
    return normalized.replace(/^indossa /, 'usa ');
}
```

**Perché un alias nel normalizer e non nel verb set del sistema item:**
- Centralizza la logica in un unico posto, coerentemente con come `'i'` → `'inventario'` e `'l'` → `'guarda'`
- Non richiede modifiche al verb set (che è una regex interna al dispatcher)
- Funziona anche per futuri comandi custom che abbiano regex contenente `usa`

Risultato: `INDOSSA TUTA SPAZIALE` → `USA TUTA SPAZIALE` → attiva `item.onUse` della tuta → comportamento corretto.

### Verifica

- `npx tsc --noEmit` → zero errori ✅
- `INDOSSA TUTA SPAZIALE` → attiva la tuta come `USA TUTA SPAZIALE` ✅
- `INDOSSA` senza argomento → non modificato (non inizia con `'indossa '`) → cade nel fallback generico con messaggio sensato ✅
